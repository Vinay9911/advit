# AutoSnap: Webpage Screenshot to WhatsApp Automation

## Overview
This is a lightweight MVP tool built for the hiring assignment. It automates the process of:
1. Reading a list of contacts and URLs from a CSV.
2. Capturing a full-page screenshot of the URL.
3. Sending the screenshot to the mapped WhatsApp number.

## Requirements
- Python 3.8+
- Google Chrome installed (or Chromium)
- A valid WhatsApp account
